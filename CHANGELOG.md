# Changelog

## 0.2.0-alpha

- Added Memory, Corrupted, and Mimic Echo behavior controllers.
- Added weighted Echo type selection, Mimic delay/chase/damage configuration, new debug commands, and new advancements.
- Hardened private Echo tracking so private Echo entities are only tracked by the target player by default.

## 0.1.0

- Initial standalone Fabric implementation for Minecraft 1.21.1.
- Added bounded player recording, per-player stages, server-directed Echo replays, admin commands, localization, placeholder assets, and advancements.
